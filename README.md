# Social Data Science Basecamp Project

## Overview
This project demonstrates a complete data science workflow, from raw data processing to statistical inference, using Python.

## Technical Stack
- **Data Manipulation**: `pandas`, `numpy`
- **Visualization**: `matplotlib`, `seaborn`
- **Statistical Modeling**: `statsmodels` (OLS Regression)

## Key Tasks Performed
1. **Data Cleaning**: Handled missing values and standardized data structures.
2. **EDA**: Performed exploratory data analysis to identify distributions and correlations.
3. **Modeling**: Developed regression models to analyze relationships between social variables.
